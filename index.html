Dataset Details 

IIIT-D, an academic Institution in Delhi, India, has seven buildings on campus namely Academic, Lecture, Library, Facilities, Dining, Boys dormitory, and Girls dormitory. A smart meter is connected to each building which measures the aggregate energy consumption of the building. We release following data related to the Institute:
            
    
                
  1. Energy Data: For each building, there is a CSV  which contains five different parameters - Voltage, Current, Power, Frequency and Power Factor - at a sampling frequency of 1 minute.
    2. 
                
  2. Weather Data: This contains temperature and humidity data at one minute frequency. This data is obtained from a nearby weather station. Further, One CSV file with sunrise and sunset timings is provided too.
    3. 
                
  3. Institute Calendar: This data file contains three columns where first column contains date, second column corresponds to whether a day was working or not and the third column shows whether a day was classified as high activity period or low activity period according to academic schedule.
    4. 
            

            All the three datasets span from 10 August 2013 till 7 July 2017. A readme file is associated with each dataset for further details.
